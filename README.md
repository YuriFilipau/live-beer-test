## Flutter Version Management (FVM)

This project uses [**Flutter_gen**](https://pub.dev/packages/flutter_gen) to generate assets.

1. Install Flutter_gen:
```bash
flutter pub get
```

2. Use Flutter_gen:
```bash
dart run build_runner build
```
